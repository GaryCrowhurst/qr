## ğŸ“– Advanced Tips

### Logo Best Practices
- Use square logos (100x100px recommended)
- Ensure high contrast between logo and background
- Keep logos simple for better scanning
- Test QR code after adding logo to ensure it scans correctly

### Optimal QR Code Sizes
- **Small (200px)**: Digital displays, small labels
- **Medium (300px)**: Standard printing, posters
- **Large (500px)**: Banners, large signage, detailed prints

### Style Selection Guide
- **Square**: Best scanning reliability, professional look
- **Dots**: Modern aesthetic, good for branding
- **Rounded**: Friendly appearance, softer design

### WiFi QR Codes
When sharing classroom WiFi:
1. Select "WiFi Network" type
2. Enter exact network name (SSID)
3. Choose correct security type (usually WPA/WPA2)
4. Enter password
5. Students can scan and connect instantly!
